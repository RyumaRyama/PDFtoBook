# 使い方
このscriptはPDFをコマンドラインで操作できるコマンド**pdftk**に依存しているため，まずはpdftkのinstallが必要です。

本家のホームページからどうぞ
[https://www.pdflabs.com/tools/pdftk-server/](url)

あとはPATHが通っている場所に置いて，空白PDFも一緒に置くだけで~~実用性はないですが~~使用できます。
